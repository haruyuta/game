使い方 / エンジンに接続できないとき
  1. VOICEVOX公式サイトからアプリをインストールし、起動しておく(起動中は自動で http://localhost:50021 にAPIが立ちます)。
  2. このHTMLを python -m http.server などlocalhost経由で開くと、CORS設定不要でそのまま動きます。
  3. file://直開きやGitHub Pagesから使ってCORSエラーが出る場合は、VOICEVOX起動中に http://localhost:50021/setting を開き、Allow Originにこのページのオリジンを追加(または CORS Policy Mode を all に変更)してください。
